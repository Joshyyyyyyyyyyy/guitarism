Phase 1: The "Look & Logic" Sprint (April)
Your goal this month is to stop being afraid of CSS and start making your apps look professional.

Week 1: Tailwind & The "Static" UI (Now - April 5)

[] Day 1: Learn Tailwind Layouts. Build a simple navbar and a sidebar using only Tailwind classes (flex, grid, hidden md:block).

[] Day 2: Learn Colors & Typography. Style a "Card" component that shows a transaction (e.g., "Bought Coffee - $5").

[] Day 3: Learn Forms. Create a "Beautiful Form" with inputs for Title, Amount, and Category.

[] Day 4: Learn Responsive Design. Make your form hide the sidebar on mobile phones (sm:, md:, lg: prefixes).

[] Day 5: Practice: Re-create a simple clone of a site like Spotify or a simple Dashboard using only HTML and Tailwind.

[] Day 6: Week 1 Recap: Fix any bugs and make sure your VS Code "IntelliSense" for Tailwind is working.

Week 2: Database & "The Big CRUD" (April 6 - April 12)

[] Day 1: Migrations. Create a transactions table with columns: title, amount, type (income/expense).

[] Day 2: Eloquent Models. Learn how to use $transaction->save() to put data into the database.

[] Day 3: The "C" (Create). Build a page that saves data from your Tailwind form into the database.

[] Day 4: The "R" (Read). Show a list of all your transactions in a nice Tailwind table.

[] Day 5: The "U" (Update). Create an "Edit" button that fills the form with old data so you can change it.

[] Day 6: The "D" (Delete). Create a "Delete" button with a confirmation popup.

Week 3 & 4: Logic & Validation (April 13 - April 30)
Learn how to Validate data (e.g., "Amount must be a number").

Learn how to use Components in Tailwind to keep your code clean.

Mini-Project: A fully working "Income vs Expense" tracker where the total updates automatically.

Phase 2: The "API & Master" Sprint (May)
Your goal this month is to make your app "talk" to other things.

[] Weeks 5-6: Advanced REST API
Turn your Finance Tracker into an API.

Learn API Resources (how to clean up the JSON data).

Learn Postman to test your API like a professional developer.

[] Weeks 7-8: Authentication & Security
Learn Laravel Breeze or Sanctum.

Make it so only you can see your transactions. This is a huge requirement for 4th-year projects.

[] Week 9: Deployment & Final Polish (Ends June 1)
Learn how to put your website online (Deployment).

Clean up your code and prepare your "Portfolio" to show your professors.
